### 🌌 Về tôi - Một hành trình xuyên vũ trụ

- 🔭 Tôi là **Hoàng Ngọc**, đang du hành trong thế giới lập trình web và công nghệ.
- 🌠 Đam mê: Xây dựng giao diện đẹp như dải ngân hà, code mượt mà như chuyển động của sao.
- 🚀 Hiện đang học sâu: **React**, **Node.js**, **TypeScript**, **Tailwind** và các công nghệ vũ trụ mới.
- 📍 Base tại **Đà Nẵng, Việt Nam** – nơi tôi ngắm sao mỗi đêm.
- 💫 Mục tiêu: Tạo ra những dự án có thể "bay" xa trong vũ trụ kỹ thuật số.
